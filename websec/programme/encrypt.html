# 全加密方案

# 介绍

原理：前端JS/SDK 使用加密数据请求，后端解密得到数据,整个传输过程中数据密文。

# 方案1

前端:生成AES加密KEY,AES'KEY对称加密数据,RSA公钥加密AES'KEY,组装数据发送请求 后端:使用RSA的私钥解密得到AES'KEY,使用AES'KEY来解密得到明文数据
