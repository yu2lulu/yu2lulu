    # passiveproxy

    # 介绍

    # passiveproxy基于代理的被动扫描器,原则上被动扫描器包含：数据来源,扫描器(引擎+郭泽),数据输出。这里的扫描器只能算是扫描器(引擎+规则),

    # passiveproxy基于anyproxy项目二次开发,使用nodejs语言,anyproxy基本上把扫描器的核心功能实现了，passiveproxy只是实现了部分规则功能.

    # 目前支持如下功能:

    - 支持响应包敏感信息检测【身份证/手机号】
- 支持敏感信息目录/文件 扫描
- jsmap泄漏探测

    # 安装和运行

    git clone https://github.com/yu2lulu/passiveproxy.git
cd passiveproxy
npm install
node index.js
